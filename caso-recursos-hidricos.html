Recursos Hídricos

Exploración y caracterización de acuíferos mediante técnicas geofísicas avanzadas

Resumen del Proyecto

Este proyecto de recursos hídricos se desarrolló en el municipio de Sogamoso, Boyacá, con el objetivo de identificar y caracterizar acuíferos para abastecimiento de agua potable. Se utilizaron técnicas de tomografía eléctrica y sondeos eléctricos verticales para mapear la estructura hidrogeológica del subsuelo.

Resultado Principal: Se identificaron tres acuíferos principales con un potencial de extracción de 120 L/s, suficiente para abastecer a 15,000 habitantes.

Metodología Aplicada

1. Tomografía Eléctrica 2D

Se realizaron 18 perfiles de tomografía eléctrica con espaciado de 5 metros entre electrodos, utilizando la configuración Wenner para obtener alta resolución en la zona saturada.

Configuración de electrodos para estudios hidrogeológicos

2. Sondeos Eléctricos Verticales (SEV)

Se realizaron 12 sondeos eléctricos verticales con espaciado de AB/2 hasta 200 metros para caracterizar la estructura vertical de los acuíferos.

3. Análisis Hidrogeológico

Los datos geofísicos se integraron con información hidrogeológica regional para establecer modelos conceptuales de flujo de agua subterránea.

Conclusiones

Los estudios geofísicos han permitido identificar un potencial hídrico significativo en el municipio de Sogamoso, con tres acuíferos principales que pueden abastecer a 15,000 habitantes de manera sostenible.
